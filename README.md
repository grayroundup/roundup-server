RoundUp collects anonymous usage data to improve functionality and measure impact.
We collect:

A randomly generated install ID

Website host and page URL when a checkout page is detected

Donation interaction events (prompt shown, donate clicked, skipped, closed)

Donation amount selected

We do NOT collect:

Names

Email addresses

Payment information

Credit card numbers

Personal identifiers

All data is used solely for analytics and product improvement.
